# CNRoutingAssignment

This repository contains the Cisco Packet Tracer files and configurations created for the Computer Networks Routing Assignment.

## Files Included

* Task1_RouterBasicConfiguration.pkt
* Task2_StaticRouting.pkt
* Task3_DynamicRouting.pkt

## Task Summary

### Task 1

Basic Router Configuration

* Hostname configuration
* Console password configuration
* Login authentication

### Task 2

Static Routing

* IP addressing
* Static route configuration
* Routing verification
* Ping testing

### Task 3

Dynamic Routing

* RIP configuration
* EIGRP configuration
* Routing verification
* Connectivity testing
